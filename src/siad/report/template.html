Appendix: Methodology

This report was generated by the SIAD (Strategic Infrastructure Acceleration Detector) pipeline. Hotspots are identified by training a world model on multi-modal satellite time series and flagging persistent deviations from counterfactual neutral-weather rollouts.

Confidence Tiers

  • Structural: SAR + lights changes consistent with construction
  • Activity: Lights changes with minimal SAR signature
  • Environmental: Optical-dominated changes (likely vegetation/water)

Persistence Threshold

All hotspots exceeded the 99th percentile acceleration score for ≥2 consecutive months.
